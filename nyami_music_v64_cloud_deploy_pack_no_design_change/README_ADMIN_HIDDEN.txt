NYAMI MUSIC v47 + OWNER ADMIN HIDDEN

Что изменено:
- сайт остаётся один;
- обычные пользователи не видят кнопку Admin;
- админка находится в этом же сайте;
- вход в админку только через скрытый адрес /#admin;
- логин/пароль админа проверяются сервером;
- без админ-сессии загрузка и удаление треков невозможны;
- общий сайт остаётся для пользователей.

Запуск:
1. Распакуй архив
2. Запусти START_WINDOWS.bat
3. Открой http://127.0.0.1:5000

Админка:
http://127.0.0.1:5000/#admin

Админ по умолчанию:
login: admin
password: admin123

Как поменять админа:
в server.py измени OWNER_LOGIN и OWNER_PASSWORD
