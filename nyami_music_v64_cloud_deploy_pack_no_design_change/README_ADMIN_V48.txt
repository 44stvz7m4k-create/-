NYAMI MUSIC v48 ADMIN — v47 + owner admin

Что добавлено поверх v47:

- общий сайт остаётся для пользователей;
- обычная регистрация/вход v47 оставлены для пользователей;

2. Запусти START_WINDOWS.bat
3. Открой http://127.0.0.1:5000
4. Админка: нажми кнопку Admin или открой http://127.0.0.1:5000/#admin

Админ по умолчанию:
