NYAMI MUSIC v47 — external sources

Что добавлено:
- в админке появился выбор источника;
- Upload file: файл грузится на наш сервер и играет в основном плеере;
- Direct audio URL: прямая ссылка на аудио играет в основном плеере;
- YouTube link: создаётся внешняя карточка с переходом на YouTube;
- Spotify link: создаётся внешняя карточка с переходом на Spotify;
- загруженные и direct-треки появляются в обычном списке сайта;
- YouTube/Spotify появляются отдельным блоком External library;
- удаление из админки удаляет источник с сайта;
- тёмная тема и UA/EN в админке сохранены;
- админка остаётся скрытой через /#admin.

Важно:
- YouTube/Spotify не скачиваются и не стримятся через наш плеер;
- они добавляются как внешние карточки с переходом по ссылке;
- Direct URL должен быть прямой ссылкой на аудиофайл.

Запуск:
1. Распакуй архив
2. Запусти START_WINDOWS.bat
3. Открой сайт
