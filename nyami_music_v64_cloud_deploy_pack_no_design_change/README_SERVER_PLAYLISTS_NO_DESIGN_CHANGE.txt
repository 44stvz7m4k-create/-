NYAMI MUSIC v47 — Server playlists, no design change

Дизайн основного сайта не менялся:
- style.css не изменялся;
- layout не переписан;
- карточки, меню, профиль, нижний и правый плееры не менялись визуально;
- добавлен отдельный слой server-playlists-v58.js/css.

Что добавлено:
- серверные плейлисты для авторизованных пользователей;
- плейлисты сохраняются в SQLite;
- после перезапуска сервера плейлисты остаются;
- поддерживаются обычные треки, загруженные треки, direct URL и внешние universal-треки;
- создание плейлиста через текущую модалку синхронизируется с сервером;
- добавление трека через текущее меню ⋯ синхронизируется с сервером;
- удаление трека из активного плейлиста синхронизируется;
- удаление плейлиста синхронизируется.

Новые endpoints:
GET /api/playlists
GET /api/playlists/{id}
POST /api/playlists
POST /api/playlists/{id}/tracks
DELETE /api/playlists/{id}/tracks?key=TRACK_KEY
DELETE /api/playlists/{id}

Важно:
- гостям серверные плейлисты недоступны;
- нужно войти обычным аккаунтом, Google или Apple;
- локальные плейлисты гостя не ломаются, но не синхронизируются.

Запуск:
1. Распакуй архив
2. Запусти START_WINDOWS.bat
3. Открой сайт

Админка:
http://127.0.0.1:5000/#admin
