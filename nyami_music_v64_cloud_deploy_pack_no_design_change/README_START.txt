NYAMI MUSIC v47 — рабочая авторизация

Запуск:
1. Распакуй архив.
2. Запусти START_WINDOWS.bat
   или вручную:
   python server.py
3. Открой:
   http://127.0.0.1:5000

Важно:
- НЕ открывай index.html двойным кликом.
- В v47 auth сделан отдельными файлами:
  auth-v47.css
  auth-v47.js
- Старые auth-экраны удалены из DOM.
- CSS/JS подключены с ?v=47, плюс сервер отдаёт их без кеша.

Что работает:
- поля ввода активные;
- видимые focus/hover/error состояния;
- регистрация через Email;
- регистрация через ник;
- регистрация через телефон;
- вход только по тем данным и паролю, которые были при регистрации;
- гостевой режим с ограничениями;
- сессия через HttpOnly cookie;
- backend Python stdlib + SQLite + PBKDF2.
